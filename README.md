## 本地构建

如果您想在本地构建并预览文档，请按照以下步骤操作：

1. 安装必要的工具:
   ```
   pip install mkdocs mkdocs-material
   ```

2. 克隆此仓库:
   ```
   git clone https://github.com/haitao926/aitoolkit_base_docs.git
   ```

3. 进入项目目录并启动本地服务器:
   ```
   cd aitoolkit_base_docs
   mkdocs serve
   ```

4. 在浏览器中访问 `http://127.0.0.1:8000` 预览文档
